What is MITRE ATT&CK?

The MITRE ATT&CK Framework is a comprehensive and globally accessible knowledge base that tracks adversary behavior across the entire attack lifecycle. Unlike simpler models like the Cyber Kill Chain, it provides a highly detailed matrix of specific tactics and techniques used by threat actors, helping organizations identify attack types and define risks more effectively.

  • Comprehensive Matrix: A large database of tactics and techniques.
  • Tracks Attack Lifecycle: Covers the full scope of an attack from start to finish.
  • Continuously Updated: A curated and reliable source of up-to-date threat information.
  • Dual Perspective: Indexes attacks from both the attacker's and defender's viewpoints.

How Security Teams Use the Framework

Red Teams (Offensive)

Replicate the documented attack scenarios to test an organization's defenses and identify weaknesses.

Blue Teams (Defensive)

Test their security controls against known adversary behaviors to strengthen their security posture.

Security Operations Teams

Use the framework to better understand adversary motivations and link their actions to specific defense classes.

The ATT&CK Matrices

The framework is divided into three main matrices, each tailored to a different environment.

ATT&CK for Enterprise

Covers adversary behavior in traditional IT networks, including platforms like Windows, macOS, Linux, and Cloud environments.

ATT&CK for Mobile

Focuses on tactics and techniques used against mobile devices running Android and iOS.

ATT&CK for ICS

Tailored to operational technology (OT) environments, covering threats to industrial control systems and manufacturing floors.

ATT&CK for Enterprise Tactics

This is the most detailed matrix, with tactics representing the "why" of an adversary's action.
